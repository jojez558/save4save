## 🚀 Free Deployment (3 Ways)

### ⭐ Option 1 — GitHub Pages (Fully Automatic, 100% Free)

1. **Create a GitHub account** → [github.com/signup](https://github.com/signup)
2. **Create a new repository** called `save4save` (make it Public)
3. **Upload all files** in this folder to the repo
4. Go to **Settings → Pages → Source → GitHub Actions**
5. The workflow in `.github/workflows/deploy.yml` runs automatically
6. Your site goes live at: `https://yourusername.github.io/save4save`

> Push any change → site updates in ~1 minute automatically!

---

### Option 2 — Netlify (Drag & Drop, No Account Needed)

1. Go to **[app.netlify.com](https://app.netlify.com)**
2. Drag this entire folder onto the page
3. Done! You get `https://random-name.netlify.app`
4. Optional: rename to `save4save.netlify.app` in Settings

---

### Option 3 — Vercel

1. Go to **[vercel.com](https://vercel.com)** → Sign up free
2. Click **Add New → Project → Upload**
3. Upload this folder
4. Done! Free `https://save4save.vercel.app`
